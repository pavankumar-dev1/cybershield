# 🛡 CyberShield

## Project Overview
CyberShield is a cybersecurity project that helps users detect risks and improve online safety.

## Setup

Install:
pip install flask flask-cors

Run:
python app.py

Open:
http://localhost:5000

## Features
- Password strength checker
- Phishing detector
- WiFi risk analyzer
- App permission checker
- Digital footprint scan
